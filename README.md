# MyNote

A simple and lightweight **text editor built with Python and Tkinter**.

MyNote allows you to create, open, edit, and save text files quickly with a clean and minimal interface.

---

## Features

* Create new files
* Open `.txt` files
* Save files
* Save files as a new file
* Undo and redo changes
* Cut, copy, and paste
* Select all text
* Warning for unsaved changes
* Simple and fast interface

---

## Interface

The application includes:

* Main text editing area
* Menu bar with:

  * **File**
  * **Edit**
  * **Help**
* Scrollbar
* Keyboard shortcut support

---

## Keyboard Shortcuts

| Shortcut           | Action     |
| ------------------ | ---------- |
| `Ctrl + N`         | New file   |
| `Ctrl + O`         | Open file  |
| `Ctrl + S`         | Save       |
| `Ctrl + Shift + S` | Save As    |
| `Ctrl + Z`         | Undo       |
| `Ctrl + Y`         | Redo       |
| `Ctrl + X`         | Cut        |
| `Ctrl + C`         | Copy       |
| `Ctrl + V`         | Paste      |
| `Ctrl + A`         | Select All |

---

## Requirements

* Python **3.10+**
* Tkinter (usually included with Python)

Check your Python version:

```bash
python --version
```

---

## How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/mynote.git
```

2. Enter the project folder

```bash
cd mynote
```

3. Run the application

```bash
python main.py
```
